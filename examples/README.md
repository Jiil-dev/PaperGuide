# Examples

This directory contains example guidebooks generated by PaperGuide.

## attention_is_all_you_need_mini.md

A complete 3-part guidebook for the **"Attention Is All You Need"** paper (Vaswani et al., 2017), generated from a mini version containing only the Abstract and Introduction sections.

**Statistics**:
- ~2,000 lines of Korean explanation
- 240 KB Markdown file
- Generated with 86 Claude Code calls (~96 minutes runtime)
- Maximum header depth: Level 5 (book-style, not specification-style)

**What to look for**:
- **Part 1** (lines 1–36): Big-picture summary of the paper
- **Part 2** (lines 37–288): Section-by-section author-perspective explanation
  - Notice how the author's word choices (`complex`, `simple`, `solely`, `entirely`) are analyzed as deliberate rhetorical devices
  - Notice how the trade-offs of removing recurrence (e.g., needing positional encoding) are explicitly discussed
- **Part 3** (lines 289–1983): Standalone deep explanations of 17 prerequisite topics
  - RNN/LSTM/GRU, CNN basics, attention history, encoder-decoder, optimization, and more
  - Each topic includes: definition → intuition → math → numerical example → connection to the paper

This sample was generated with `--phase 3 --mode cache --cache-dir samples/_tmp_phase3/cache`.

## Generating your own examples

```bash
.venv/bin/python -m src.main \
    --input data/papers/your_paper \
    --output examples/your_paper_guidebook.md \
    --mode live \
    --cache-dir data/cache_your_paper \
    --phase 3
```

See the [main README](../README.md) for full instructions.
